# WorkLedger User Flow

## Primary 10-Second Record Flow

```text
앱 실행
  -> 홈/오늘 기록
  -> 오늘 WorkRecord 조회
  -> 출근 전이면 [출근하기] 탭
  -> 근무 중이면 [퇴근하기] 탭
  -> 퇴근 후이면 오늘 기록 완료 상태 확인
  -> 현재 시각 저장
  -> 기록된 시각 표시
```

## Notification Record Flow

```text
설정/알림 권한
  -> 상시 알림 켜기
  -> Android 알림 권한 확인
  -> 상시 알림 표시
  -> 알림 액션 [출근] 또는 [퇴근] 탭
  -> WorkRecord 저장
  -> 홈/오늘 기록에서 결과 확인
```

알림 권한이 거부되면 앱 내부 홈 기록 흐름은 유지한다.

## Edit Record Flow

```text
홈/오늘 기록
  -> 기록 수정
  -> 출근/퇴근 시각 수정
  -> 야근/퇴근 지연/휴일근무 태그 선택
  -> 메모 입력
  -> 저장
  -> 홈/오늘 기록
```

## Leave Flow

```text
홈/오늘 기록
  -> 연차 관리
  -> 기준 연도 총 연차 입력
  -> 연차 사용 추가
  -> 사용 날짜와 사용량 입력
  -> 잔여연차 계산 표시
```

잔여연차는 저장하지 않고 `총 연차 - 사용 연차 합계`로 계산한다.

## Monthly Summary Flow

```text
홈/오늘 기록
  -> [월간 요약] 탭
  -> 현재 연월의 월간 요약 화면
  -> 필요 시 월 선택
  -> 월간 근무시간 확인
  -> 근무일 확인
  -> 초과 참고 시간 확인
  -> 이번 달 기록 목록 확인
  -> incomplete 기록 상태 확인
```

초과근무는 개인 참고용이며 임금 정확성을 보장하지 않는다.
월간 요약 화면은 근무 합계, 초과 참고, 태그별 참고, 연차 사용/잔여 요약을 함께 표시한다.

## Calendar View Flow

```text
홈/오늘 기록
  -> 퇴근 후 [달력 보기] 탭
  -> 현재 연월의 달력 보기 화면
  -> 날짜별 완료, 출근만 기록, 시간 누락, 기록 없음 상태 확인
  -> 날짜 선택
  -> 선택 날짜의 출근/퇴근 시간, 총 근무시간, 태그, 메모 확인
  -> 선택 날짜가 오늘이면 [오늘 기록 수정]으로 기존 수정 화면 이동
```

달력 보기는 읽기 전용 조회 화면이다. 이전 날짜 새 기록 생성, 이전 기록 수정, 삭제, 월간 리포트 생성은 이 흐름에 넣지 않는다.

## Pricing Fake-Door Flow

```text
월간 요약
  -> 월간 리포트 만들기
  -> 가격표/fake-door
  -> Report Pass 또는 Pro 탭
  -> 클릭 이벤트 저장
  -> MVP 준비 중 안내
```

실제 결제와 실제 PDF/CSV 생성은 하지 않는다.

## Failure Flow

```text
사용자 액션
  -> 검증 실패 또는 저장 실패
  -> 명확한 에러 표시
  -> 기존 입력값 유지
  -> 사용자가 수정 후 재시도
```

권한 거부나 저장 실패는 조용히 무시하지 않는다.
